# VueBlog

这是一个使用Vue 3和Vite构建的个人博客网站。

## 特点

- 使用Vue 3和Vite构建的现代化前端应用
- 响应式设计，适配各种设备屏幕
- Markdown文章支持，包括代码高亮和数学公式
- 分类和标签系统，方便内容组织
- 搜索功能，快速查找内容
- 暗色/亮色主题切换

## 技术栈

- Vue 3 (组合式API)
- Vite
- Vue Router
- Pinia状态管理
- Bootstrap 5
- KaTeX (数学公式渲染)
- Highlight.js (代码高亮)
- FontAwesome图标
